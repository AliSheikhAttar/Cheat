## Shell


### info
- Information about ls. use “q” to exit
```bash
info ls
```
- Complete information about ls
```bash
man ls
```

- help
```bash
<command> --help
```

### History

```bash
history
history <n> #last <n> commands
```
- run previous commands
```bash
!! # previous command
!!<n> # nth previous command
```
- delete history
```bash
history -c; history -w;
```
